# Language Labeler

LanguageLabeler is a browser extension designed to help your jobhunting experience on LinkedIn by labeling at the top of job postings which languages are mentioned in the description.

## Usage

### Setting up:
1. Download the repository.
2. In Firefox, go to the url `about:debugging`
3. Select `This Firefox`
4. Select `Load Temporary Add-on...`
5. To load, select the `manifest.json` file.

The add-on is now added to this session of Firefox and will remain until it is removed or the browser is closed.

### Getting languages listed

1. Navigate to LinkedIn jobs section.
2. When you are on a page that lists jobs, such as the search, refresh the page to activate the add-on.
